# How to setup
### Archlinux
```bash
sudo pacman -Sy
sudo pacman -S nodejs git
```

### Ubuntu/Debian
```bash
sudo apt update -y
sudo apt install git nodejs
```

### MacOS
```bash
brew install git node
```
### Windows (WSL)
```bash
wsl --install
```
(restart and open ubuntu terminal, use Ubuntu/Debian commands)

## Run
```bash
git clone https://github.com/itzAkss/taupe.git
cd taupe
npm install
npm start
```
